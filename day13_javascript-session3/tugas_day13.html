Manipulasi String

Di sini Anda dapat melihat contoh manipulasi string menggunakan metode JavaScript seperti `toUpperCase()`, `toLowerCase()` dan `replace()`. Manipulasi string ini memungkinkan kita untuk merubah format teks sesuai kebutuhan.

Metode Array

Array adalah salah satu struktur data yang paling berguna dalam JavaScript. Anda bisa menambahkan, menghapus, dan memanipulasi data dengan berbagai metode seperti `push()`, `pop()`, dan `join()`.

Arrow Function

Arrow function adalah cara singkat untuk menulis fungsi dalam JavaScript. Ini sering digunakan untuk menulis kode yang lebih ringkas dan mudah dipahami dalam berbagai operasi JavaScript.

Object Conditional

Object conditional digunakan untuk mengecek kondisi pada objek secara dinamis. Di sini kita dapat mengetahui status dari properti tertentu.
